# Ping-pong

Это игра-прототип, где два игрока на одном ноутбуке/компьютере могут играть в настольный тенис.

Суть игры:
Мячь летит в одну из сторон, где находится какой-то игрок. Игрок к которому движется мяч должен ему перегарадить путь к краю окна, а иначе он проиграет. После столкновения ракетки(игрока) и мяча - мяч полетит в другую сторону, то-есть к другому игроку.

Проигрывает тот, кто не смог поймать мяч.

Игра находится в функциональном прототипе. Это значит, что функционал указанный выше будет работать, но интерфейс будет не насыщенным.
